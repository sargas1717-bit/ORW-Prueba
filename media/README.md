# 📁 Carpeta de Medios — WRO Vision Pro

Coloca aquí tus imágenes y videos de prueba para analizarlos con el sistema de visión.

## Formatos Soportados

| Tipo    | Extensiones               |
|---------|---------------------------|
| Imagen  | `.jpg`, `.jpeg`, `.png`, `.gif`, `.webp` |
| Video   | `.mp4`, `.webm`, `.mov`   |

## Cómo usarlos

1. Copia tus archivos de imagen o video en esta carpeta.
2. Abre el sistema en el navegador (`index.html`).
3. Haz clic en el botón **📁 Subir Foto/Video** para cargar el archivo.
4. El sistema analizará el contenido en tiempo real con tu calibración actual.

## Sugerencias

- Usa videos de la pista WRO grabados desde la misma altura y ángulo que la cámara del robot.
- Ayuda a calibrar los colores antes de una competencia.
